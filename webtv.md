# BUT2 - Web TV

Évaluation de la Web TV :
Sur 80 points

| criteria | points |
| -- | -- |
| Présence d'un écran d'attente | 1 point |
| Le logo de la chaine est présent | 1 point |
| Des jingles audio et vidéo permettent la transition entre les rubriques | 3 points |
| Les < lower third > identifient les invités | 1 point |
| L'émission possède un générique de début et de fin | 3 points |
| Le motion design ajoute du dynamisme aux créations graphiques | 4 points |

La structure du cahier des charges est respectée :

| criteria | points |
| -- | -- |
| Durée entre 20 min et 30 min | 2 points |
| 3 chroniques minimum | 2 points |
| 1 chroniqueur canadien par chronique | 2 points |
| 1 chronique en anglais | 2 points |
